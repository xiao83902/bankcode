# BankCode
臺灣金融機構分行代碼查詢工具
快速查詢台灣各銀行分行代碼的命令列工具，支援模糊搜尋、銀行簡稱、一鍵複製等功能。
✨ 功能特色
🔍 模糊搜尋 - 輸入「羅東一銀」即可找到「第一銀行羅東分行」
🏷️ 銀行簡稱 - 支援常用簡稱如「一銀」「合庫」「中信」「玉山」等
🔢 代碼搜尋 - 輸入 7 碼分行代碼直接查詢
🌾 農漁會支援 - 輸入「五結農會二結」找到「五結鄉農會二結分部」
📋 一鍵複製 - 搜尋後輸入編號即可複製分行代碼

`板橋合庫`	合作金庫	合庫板橋分行
`中信宜蘭`	中國信託	中信宜蘭分行
`0172288`	7碼代碼查詢	兆豐銀行羅東分行
`007`	3碼總行代碼	第一銀行所有分行
`羅東`	地區查詢	羅東地區所有銀行
`五結農會二結`	農會模糊搜尋	五結鄉農會二結分部
`蘇澳漁會`	漁會查詢	蘇澳區漁會所有據點
